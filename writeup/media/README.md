# Media Files for Kaggle Writeup

This directory should contain the following media files for your Kaggle submission:

## Required Files

1. **cover.png** (or .jpg)
   - Cover image for your writeup
   - Recommended size: 1200x600 pixels
   - Should be visually appealing and represent your project

2. **pipeline_diagram.png**
   - Diagram showing the training pipeline
   - Should include: Data → Model → GRPO → Reward → Output
   - Can be created with tools like draw.io, Mermaid, or PowerPoint

3. **examples.png**
   - Screenshots or examples of model outputs
   - Show before/after comparisons
   - Include reasoning traces and answers

## Optional Files

- **training_curves.png**: Training loss and reward curves
- **architecture.png**: Detailed architecture diagram
- **comparison.png**: Comparison with baseline models

## Creating the Images

### Cover Image
- Use a design tool (Canva, Figma, etc.)
- Include project title
- Add relevant visuals (brain, reasoning, etc.)
- Keep it professional and clean

### Pipeline Diagram
- Show data flow
- Highlight key components (GRPO, rewards)
- Use clear labels and arrows
- Keep it simple and readable

### Examples
- Take screenshots of actual model outputs
- Show the `<reasoning>` and `<answer>` format
- Include diverse examples (math, science, etc.)
- Add annotations if helpful

## Tips

- Use consistent color scheme
- Ensure high resolution (at least 300 DPI for print)
- Keep file sizes reasonable (< 5MB each)
- Use PNG format for diagrams, JPG for photos
